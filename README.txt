Pay with Solana Pay for WooCommerce. Decentralized, permissionless, and fast payment gateway powered by Solana. Accept payments in SOL, USDC, USDT and more.

== Description ==

This plugin adds Solana Pay to your WooCommerce store and lets you accept payments in SOL, USDC, USDT and more. It supports payments via **QR Codes** and through all major Solana wallets like Phantom, Solflare, or Backpack.

It adds a Solana Pay express checkout button to the Checkout page which brings up a popup window when clicked. The popup window handles all payment transactions giving your customers a single, seamless checkout experience.

A Quick and easy way to add crypto payments via Solana to your website. Give your customers a fast crypto checkout experience and increase your conversion rate.

= Released features =

Currently,  **Pay with Solana Pay for WooCommerce** provides the following features:

* No hidden commission. Payments go directly into the Solana wallet address that you configure.
* Solana Pay checkout button on the Checkout page.
* Set your own RPC endpoint for fast, uncongested payment experience for your customers.
* Supports payments in SOL, USDC and USDT. More tokens are planned for future releases.
* Language support for: English, German, Turkish and more to come.

== Installation ==

= Requirements =

To install **Pay with Solana Pay for WooCommerce**, you need:

* WordPress Version 5.2 or newer (installed)
* WooCommerce Version 3.0 or newer (installed and activated)
* PHP Version 7.2 or newer
* BCMath extension for PHP installed

= Instructions =

1. Log in to WordPress admin.
2. Go to **Plugins > Add New**.
3. Search for the **Pay with Solana Pay for WooCommerce** plugin.
4. Click on **Install Now** and wait until the plugin is installed successfully.
5. Click on **Activate**. (You can also activate on the **Plugins > Installed Plugins** page).

= Setup Configurations =

1. After you have activated the plugin, go to **WooCommerce  > Settings**.
2. Click the **Payments** tab.
3. Click on **Solana Pay**.
4. Click the **Enable Solana Pay** checkbox.
5. Add your wallet address to the **Merchant Wallet Address** field.
6. Add your RPC Endpoint URL to the **RPC Endpoint** field.
7. Enable Solana Tokens you will like to accept and adjust the **Exchange Rate** column to the appropriate rates you prefer.
8. After you have successfully tried out in Test mode, remove the check on the **Enable Test Mode** checkbox to use Production Mode.
9. Click **Save changes**.

== Frequently Asked Questions ==

= What is Solana Pay? =

Solana Pay is a fast, decentralized, permissionless, and open source payment framework built on Solana blockchain. It is built for immediate transactions with fees that are fractions of a penny, and a net-zero environmental impact. See [Solana Pay](https://solanapay.com/) to learn more.

= Which Solana tokens can be accepted as payment? =

**Pay with Solana Pay for WooCommerce** plugin currently supports payments in SOL, USDC and USDT. Support for more tokens are planned to be added in the future.
